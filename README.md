# Продажа квартир в Санкт-Петербурге — анализ рынка недвижимости
**Задачи проекта:** Используя данные сервиса Яндекс.Недвижимость, определить рыночную стоимость объектов недвижимости и типичные параметры квартир.

**Описание проекта:** На основе данных сервиса Яндекс.Недвижимость определена рыночная стоимость объектов недвижимости разного типа, типичные параметры квартир, в зависимости отудаленности от центра. Проведена предобработка данных. Добавлены новые данные. Построены гистограммы, боксплоты, диаграммы рассеивания.

https://datalens.ru/hprcmhws37223 — Итоговая визуализация данных со стоимостью объектов недвижимости разного типа, полученная на основе данных сервиса Яндекс.Недвижимость.

Используемые инструменты: DBeaver, Yandex DataLens.

# Игра "Секреты Темнолесья" - анализ продаж внутриигровой валюты
**Задачи проекта:** Используя базу данных онлайн-игры "Секреты Темнолесья", определить долю платящих игроков и проанализировать, кто совершает покупку внутриигровых предметов.

**Описание проекта:** На основе данных онлайн-игры "Секреты Темнолесья" определена группа игроков, совершающая наибольшее количество покупок, проанализирована структура продаж внутри игры, выявлены аномалии в продажах внутриигровых предметов. По итогам анализа составлена аналиическая записка с рекомендациями для заказчика. 

Используемые инструменты: DBeaver.

# Анализ рынка инвестиций для финансовой компании — анализ исторических данных для анализа перспектив финансирования стартапов
**Задачи проекта:** Используя исторические данные об инвестициях в стартапы, исследовать динамику и структуру финансирования стартапов для разработки инвестиционной стратегии.

**Описание проекта:** На основе исторических данных определены наиболее дорогостоящие стартапы по категориям, выявлена жизнеспособность стартапов и установлена оптимальная стоимость стартапа на рынке.

Используемые инструменты: Jupyter Notebook.
